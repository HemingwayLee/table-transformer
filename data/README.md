# Data source

# Folder structure 
```
data
├── images_filelist.txt
├── train_filelist.txt
├── val_filelist.txt
├── images
├── train
└── val
```

* train and val images should be in `data/images` folder
* `*.xml` are in the `data/val` and `data/train` folder
* `*_filelist.txt` files are in `data` folder
